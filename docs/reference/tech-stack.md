# Technology stack

| Technology | Role |
|------------|------|
| **Lit** | Web components (core and extensions). |
| **TypeScript** | Typed API and implementation. |
| **WebAwesome** | UI primitives and icons. |
| **Vite** | Build and dev server. |
| **Monaco** | Code editor (via extension). |

Other extensions add: Pyodide, WebLLM, RxDB, Xenova transformers, etc.
